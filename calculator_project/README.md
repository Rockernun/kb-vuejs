- [ ] 숫자를 클릭하면 클릭한 숫자를 출력 칸에 표시해야 한다. 예를 들어, 1을 클릭하면 1이 출력되고, 1과 2를 연속으로 클릭하면 두 숫자를 조합해 12가 출력된다. 3, 4, 5 숫자 3개를 연속해서 클릭하면 세 자리 숫자 345가 출력된다.

- [ ] 숫자가 표시된 상태에서 사칙연산 기호를 클릭하면 출력칸에 표시된 숫자가 사라진다.

- [ ] 다시 한번 숫자를 클릭하면 1번 처럼 입력한 숫자가 출력칸에 표시된다. 예를 들어, 3과 0을 연속해서 클릭하면 30이 출력된다.

- [ ] 숫자가 출력칸에 표시된 상태에서 등호를 클릭하면 지금까지 입력한 숫자의 연산 결과가 화면에 보인다. 예를 들어, 처음 임력한 12, 그 다음 입력한 +, 마지막 입력한 30이 하나의 수식으로 연결되어 12 + 30을 연산한 결과인 42가 출력되는 것이다. 여기서 등호가 아닌 다른 기호를 클릭하면 등호를 클릭할 때까지 2번과 3번 과정을 반복한다.

- [ ] 다시 등호를 클릭하면 아무런 일도 일어나지 않는다.

- [ ] 새로운 숫자를 클릭하면 연산 결과 뒤로 클릭한 숫자가 붙습니다. 예를 들어, 현재 42가 출력된 상태에서 5를 클릭하면 425가 출력된다.

- [ ] 사칙 연산 기호를 클릭하면 2번 과정부터 반복한다.
